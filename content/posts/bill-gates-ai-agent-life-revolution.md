# 比尔·盖茨：AI Agent将彻底改变人类生活方式

即使到了今天，我对软件的热爱仍如同当年我和保罗·艾伦一起创立微软时一样浓烈。尽管软件在过去几十年里取得了显著的进步，但是，从很多方面来看，它依然有些 “笨拙”。

在计算机上完成任务时，你需要告诉设备使用哪个应用程序。例如，你可以用微软 Word 和谷歌文档撰写商业提案，但它们无法帮你发送邮件、分享自拍、分析数据、策划派对或购买电影票。

即便是最优秀的网站，也只能片面地了解你的工作、个人生活、兴趣及人际关系，在利用这些信息来为你服务方面能力有限。如今，要做到这些，似乎只能依靠另一个人，比如你的亲密朋友或个人助理。

但在未来五年内，这一切将彻底改变。你不再需要针对不同的任务使用不同的应用。你只需用日常语言告诉你的设备你想要做什么，软件将能够根据你愿意分享的信息量做出个性化的回应，因为它将深入理解你的生活。在不远的将来，任何联网的人都将能夠拥有一个由 AI 驱动的个人助理，其能力将远超现今的技术水平。

这种能够理解自然语言并根据对用户的了解来完成多种任务的软件，被称为 “Agent”。我对 Agent 的概念已经思考了近 30 年，在我 1995 年的著作《未来之路》中也曾提及，但直到最近，由于 AI 的飞速发展，Agent 才变得切实可行。

Agent 不仅将改变每个人与计算机的互动方式，还将颠覆整个软件产业，带来自从人们从敲击命令转变为点击图标以来计算机领域最重大的革命。

# 个人助理

一些批评家指出，软件公司曾经推出过类似的产品，但用户并没有真正接纳它们。（我们还记得微软 Office 中曾经出现的数字助理 Clippy，后来被我们弃用，它至今仍是人们的笑柄。）那么，为什么人们会使用 Agent 呢？

原因在于，新一代的 Agent 将远远优于以往。你将能与它们进行更为微妙的交流。它们将更加个性化，并且不再局限于完成像写信这样的简单任务。Clippy 与这些 Agent 的关系，就像旋转式电话与智能手机的关系一样。

Agent 可以在你愿意的情况下帮助你处理所有事务。它将通过获得跟踪你的在线互动和真实世界位置的权限，深入理解你参与的人、地点和活动。它将洞悉你的个人和工作关系、爱好、偏好及日程。你可以选择它何时介入帮助你处理某事或提醒你做决策。

“Clippy 是一个机器人，不是 Agent。”

要理解 Agent 将带来的巨大变革，让我们将其与当今的 AI 工具进行对比。大多数这类工具是机器人，它们只限于单一应用，通常只在你输入特定单词或寻求帮助时才会介入。由于它们无法记住你上一次的使用方式，因此无法学习你的偏好或变得更加高效。Clippy 就是这样一个机器人，而不是 Agent。

Agent 则更为智能。它们是主动的 —— 能够在你提出要求之前主动提出建议。它们能够跨应用完成任务，并且随着时间的推移，因为记住了你的活动并识别你行为中的意图和模式，它们会不断改进。根据这些信息，它们会提供它们认为你需要的东西，尽管最终决定权始终掌握在你手中。

想象一下你想规划一次旅行。一个旅行机器人可能会帮你找到合适的酒店。而 Agent 则会知道你计划在哪个季节旅行，根据它对你是否总是尝试新地点还是喜欢重复访问相同地点的了解，它可以提出地点建议。在被询问时，它还会根据你的兴趣和冒险倾向推荐活动，并在你喜欢的餐馆预订位置。如果你今天想要这样深度个性化的规划，你需要支付旅行代理的费用并花时间告诉他们你的需求。

AI Agent 最令人兴奋的影响，在于它们将使得今天对大多数人来说过于昂贵的服务变得普及。它们将在医疗保健、教育、生产力以及娱乐和购物等四个领域产生巨大的影响。

# 医疗保健

当前，AI 在医疗保健领域的主要角色是协助处理行政性任务。例如，Abridge、Nuance DAX 和 Nabla Copilot 等工具可以在就诊时捕获音频，然后为医生编写笔记供其查看。

真正的改变将在 Agent 能够帮助病人进行初步的分诊、提供如何处理健康问题的建议，以及决定他们是否需要寻求治疗时发生。这些 Agent 还将协助医疗工作者做出决策，提高工作效率。（比如，像 Glass Health 这样的应用已能分析病人的概况，并向医生提供可能的诊断建议。）这对于贫穷国家的人们来说尤其有益，因为那里的许多人根本没机会看医生。

这些临床 Agent 推出的速度会比其他类型的 Agent 慢，因为正确处理每一件事关系到生命安全。人们需要看到证据，表明健康 Agent 总体上是有益的，即使它们不完美且会犯错。当然，人类也会犯错，而且无法获得医疗护理同样是一个问题。

“美国有一半需要心理健康护理的退伍军人得不到帮助。”

心理健康护理是另一个例子，展示了 Agent 将使这项服务几乎对所有人都可用。如今，每周一次的心理治疗似乎是一种奢望。但实际上存在大量未被满足的需求，许多本可以从治疗中获益的人无法得到这种服务。例如，RAND 研究发现，美国有一半需要心理健康护理的退伍军人并未得到相应的帮助。

受过良好心理健康培训的 AI Agent 将使心理治疗变得更加经济实惠和易于获得。Wysa 和 Youper 是这方面的早期例子。但 Agent 的作用将更加深入。如果你选择向心理健康 Agent 分享足够的信息，它将了解你的生活过程和人际关系。当你需要时，它们随时可用，且永远不会不耐烦。它甚至可以在你的允许下，通过智能手表监测你在治疗过程中的生理反应 —— 例如，当你谈论与老板的问题时心跳加速 —— 并建议你何时应该去见人类治疗师。

# 教育

多年来，我一直对软件简化教师工作和帮助学生学习的各种方式感到兴奋。软件不会取代教师，但会辅助他们的工作 —— 为学生提供个性化学习并解放教师，让他们从繁琐的文书工作中摆脱出来，从而有更多时间专注于教育工作的最重要部分。这些变化终于开始以显著的方式发生。

目前的最新技术是 Khan Academy 创造的基于文本的机器人 Khanmigo。它可以在数学、科学和人文科学领域为学生提供辅导 —— 例如，它能解释二次方程公式，并创造数学练习题。它还能帮助教师编写教案等。我长期以来一直是 Sal Khan 的粉丝和支持者，最近我在我的播客中邀请他讨论教育和 AI。

但基于文本的机器人只是第一波。Agent 将为学习开启更多的机会。

比如，很少有家庭能支付起孩子一对一的家教补课费用。如果 Agent 能够捕捉到高效家教的关键因素，它们将为所有需要的人提供这种补充教学。如果一个辅导 Agent 知道一个孩子喜欢 Minecraft 和泰勒·斯威夫特，它会利用 Minecraft 教授他们计算形状的体积和面积，用泰勒的歌词教他们讲故事和押韵模式。与当今的基于文本的辅导相比，这种体验将更加丰富 —— 例如，包含图形和声音 —— 并更加个性化。

# 生产力

当前，这个领域已经存在激烈的竞争。例如，微软正在将其 Copilot 功能集成到 Word、Excel、Outlook 以及其他服务中。谷歌也在用 Assistant、Bard 及其生产力工具做着类似的事情。这些协助工具能做很多事情，如将书面文档转换为幻灯片，使用自然语言回答有关电子表格的问题，以及在代表每个人观点的同时总结电子邮件线索。

Agent 将做得更多。拥有一个 Agent 就像有一个人专门致力于帮助你处理各种任务，如果你愿意，它们还可以独立完成这些任务。如果你有一个商业想法，Agent 将帮助你撰写商业计划，制作演示文稿，甚至生成你的产品可能的外观形状。公司将能够让员工直接咨询 Agent，并在每次会议中利用它们来回答问题。

“如果你的朋友刚做了手术，你的 Agent 将主动提出送花，并能为你下单。”

无论你是否在办公室工作，你的 Agent 都能以类似于现在个人助理支持高管的方式来帮助你。如果你的朋友刚做了手术，你的 Agent 将主动提出送花，并能为你下单。如果你告诉它你想和大学时代的室友见面，它将与他们的 Agent 协作，找到合适的时间，临近会面时，它还会提醒你，他们的大孩子刚开始在当地大学上学。

# 娱乐与购物

现在，AI 已经可以帮助你选择新电视，推荐电影、书籍、节目和播客。例如，我投资的一家公司最近推出了 Pix，它允许你提出问题（比如 “我会喜欢哪些罗伯特・雷德福的电影，我在哪里可以看？”），然后根据你过去的喜好做出推荐。Spotify 拥有一个 AI 驱动的 DJ，不仅根据你的偏好播放歌曲，而且还可以和你交谈，甚至能称呼你的名字。

Agent 不仅会做出推荐，它们还会帮助你采取行动。如果你想购买相机，你可以让你的 Agent 为你阅读所有评论，总结它们，提出推荐，并在你做出决定后下订单。如果你告诉你的 Agent 你想看《星球大战》，它会知道你是否已订阅了相应的流媒体服务，如果没有，它会提出帮你注册。如果你不确定想看什么，它会提供个性化建议，然后找出如何播放你选择的电影或节目。

你还将能够获得根据你的兴趣定制的新闻和娱乐内容。CurioAI 能够根据你提出的任何主题创建定制播客，这只是未来的一个预览。

# 科技行业的巨大冲击

简而言之，Agent 将能够帮助处理几乎任何活动和生活的任何领域。这将对软件业务和社会产生深远的影响。

在计算行业，我们谈论的是平台 —— 应用程序和服务构建在其上的技术。Android、iOS 和 Windows 都是平台。Agent 将成为下一个平台。

“要创建一个新的应用或服务，你只需告诉你的 Agent 你想要什么。”

要创建一个新的应用或服务，你不需要了解如何写代码或进行图形设计。你只需告诉你的 Agent 你想要什么。它将能够编写代码，设计应用的外观和感觉，创建标志，并将应用发布到在线商店。OpenAI 本周推出的 GPT 系列产品就是一个未来的预览，即非开发者也可以轻松创建和分享他们自己的助手。

Agent 将影响我们使用软件的方式以及软件的编写方式。它们将取代搜索网站，因为它们在查找信息并为你总结信息方面更为出色。它们将取代许多电子商务网站，因为它们能为你找到最佳价格，而且不局限于少数几个供应商。它们将取代文字处理器、电子表格和其他生产力应用程序。今天是独立的业务 —— 搜索广告、带广告的社交网络、购物、生产力软件 —— 将成为一个业务。

我不认为任何一个公司会主导 Agent 业务 —— 将有许多不同的 AI 引擎可用。今天，Agent 嵌入在诸如文字处理器和电子表格等其他软件中，但最终它们将独立运作。尽管一些 Agent 将免费使用（并通过广告支持），但我认为你将为大多数 Agent 付费，这意味着公司将有动力使 Agent 为你工作，而不是为广告商工作。如果今年开始研究 AI 的公司数量有任何迹象的话，将会有异常激烈的竞争，这将使 Agent 非常便宜。

但在我所描述的复杂 Agent 成为现实之前，我们需要面对关于技术及其使用方式的许多问题。我之前写过 AI 带来的问题，所以在这里我将特别关注 Agent。

# 技术挑战

目前还没有人弄清楚 Agent 的数据结构应该是什么样的。为了创建个人 Agent，我们需要一种新型数据库，能够捕捉你的兴趣和关系的所有细节，并在保护你隐私的同时迅速调取信息。我们已经看到了新的信息存储方式，比如向量数据库，这可能更适合存储由机器学习模型生成的数据。

还有一个未解决的问题是，人们将与多少个 Agent 进行互动。你的个人 Agent 是否会与你的心理治疗 Agent 和你的数学家教 Agent 分开？如果是这样，你什么时候希望它们相互合作，什么时候应该各司其职？

“如果你的 Agent 需要与你联系，它会通过耳机与你说话或在你的手机上显示。”

你将如何与你的 Agent 互动？公司正在探索各种选项，包括应用程序、眼镜、挂坠、别针，甚至全息图。所有这些都有可能，但我认为人与 Agent 互动的第一个重大突破将是耳机。如果你的 Agent 需要与你联系，它会通过耳机与你说话或在你的手机上显示。（“你的航班延误了。你想等待，还是需要我帮你重新预定？”）如果你愿意，它还能监控进入你耳朵的声音，并通过屏蔽背景噪音、放大难以听清的语音或使你更容易理解带有重口音的人的讲话来增强它。

还有其他挑战。目前还没有一个标准协议，能够让 Agent 之间相互沟通。Agent 的成本需要降低，以便每个人都能负担得起。激发 Agent 给出正确答案的方式需要更加简化。我们需要避免错误信息的产生，特别是在像健康这样的领域，准确性至关重要，并确保 Agent 不会因为其偏见而伤害人们。我们不希望 Agent 做出它们不该做的事情。（虽然我对恶意 Agent 的担忧比对人类犯罪分子利用 Agent 进行恶意活动的担忧要少。）

# 隐私和其他重大问题

随着这一切的发展，网络隐私和安全问题将变得比现在更加紧迫。你会希望能够决定 Agent 可以访问哪些信息，因此你会确信你的数据只与你选择的人和公司共享。

但你与 Agent 共享的数据归谁所有，你如何确保它被适当地使用？没有人想因为与心理治疗 Agent 分享的内容而开始收到相关的广告。执法部门能否使用你的 Agent 作为对你的证据？你的 Agent 什么时候会拒绝做可能对你或他人有害的事情？谁来决定 Agent 中的价值观？

还有一个问题是 Agent 应该分享多少信息。假设你想见一个朋友：如果你的 Agent 与他们的 Agent 对话，你不希望它说，“哦，她周二要和其他朋友见面，不打算邀请你。” 如果你的 Agent 帮你撰写工作邮件，它需要知道不应使用你的个人信息或以前工作中的专有数据。

这些问题已经成为科技行业和立法者的首要考虑。我最近参加了由参议员 Chuck Schumer 组织的一次 AI 论坛，与其他科技领袖一同参加，许多美国参议员也出席了会议。我们分享了关于这些和其他问题的想法，并讨论了立法者制定强有力立法的必要性。

但其他问题不会由公司和政府决定。例如，Agent 可能会影响我们与朋友和家人的互动方式。今天，你可以通过记住他们生活中的细节来显示你关心他们 —— 比如，记住他们的生日。但当他们知道你的 Agent 可能提醒了你，并负责发送花朵时，这对他们来说是否还有意义？

在遥远的未来，Agent 甚至可能迫使人类面对关于价值的深刻问题。想象一下，Agent 变得如此出色，以至于每个人都可以在几乎不用工作的情况下享有高质量的生活。在这样的未来，人们会如何利用他们的时间？当 Agent 拥有所有答案时，还会有人想要接受教育吗？当大多数人有大量空闲时间时，你能拥有一个安全繁荣的社会吗？

但我们距离这一点还很远。与此同时，Agent 正在来临。在接下来的几年里，它们将彻底改变我们的生活方式，无论是在线上还是线下。
